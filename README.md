<h2 class="header2">Description</h2>
<p class="text-content">
Fourth Capstone Project for 100 days of python, utilizes apis to find deals using kiwi api

Steps:

- First checks the specified prices inside the google sheet,
- Crossreferences this with the kiwi API to check if there are any cheaper alternatives
- Finally notifies users who are on the mailing list about cheaper flights

PLEASE NOTE: This app requires the use of API keys to run as it was a project involving learning about API, i do not provide these keys so if you wish to test the APP you need to populate the API keys in the env file with your own
